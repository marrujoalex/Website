"MongoKitten is performant. More so than similar libraries."

For MongoKitten we've written an optimized pure-swift BSON library for MongoDB entities and their protocol.

BSON is specifically designed to consider Swift's strenghts and Swift's weaknesses performance wise.
Whilst exposing an extremely simple and Swifty API we provide top-notch performance.

MongoKitten embraces type safety and performance in combination with the language features you love about Swift.
